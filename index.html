Python

1. Variables

Introduction to variables, naming conventions, and scope in Python.

2. Data Types

  • Numeric: Includes integer (int), floating point (float), and complex types.
  • String: Represents text, enclosed in single, double, or triple quotes.
  • List: A collection of items in a sequence, mutable.
  • Tuple: A collection of ordered elements, immutable.
  • Dictionary: A collection of key-value pairs, unordered.
  • Set: A collection of unique elements, unordered.
N

3. Operators

  • Arithmetic Operators: +, -, *, /, %, //, **.
  • Comparison Operators: ==, !=, >, <, >=, <=.
  • Logical Operators: and, or, not.
  • Assignment Operators: =, +=, -=, *=, /=, %=, etc.
  • Bitwise Operators: &, |, ^, ~, <<, >>.

4. Decision Making Statements

  • if statement: Executes a block of code if the condition is true.
  • if-else statement: Executes one block if the condition is true, another if false.
  • if-elif-else ladder: Multiple conditions checked in sequence.

5. Loops

  • for loop: Loops through a sequence (like a list or range).
  • while loop: Repeats as long as the condition is true.
  • break: Exits the loop prematurely.
  • continue: Skips the current iteration of the loop and proceeds to the next one.

6. Functions

A function is a block of reusable code that performs a specific task. It can take arguments and return values.

7. Exception Handling

Handling errors in Python using try, except, else, and finally blocks.

8. File Handling

  • Open: Opens a file for reading or writing.
  • Read: Reads data from a file.
  • Write: Writes data to a file.
  • Close: Closes the file after operation.

9. Object-Oriented Programming

Introduction to classes, objects, inheritance, polymorphism, and encapsulation.

10. Modules and Packages

Modules are files containing Python definitions, and packages are collections of modules.
